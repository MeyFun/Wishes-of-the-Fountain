# Wishes of the Fountain

**Wishes of the Fountain** — это атмосферная 2D-игра для ПК, в которой вы берете на себя роль таинственного обитателя «подмирья» городского фонтана. Ваша задача — принимать монетки, брошенные людьми, и исполнять их сокровенные или повседневные желания.

## 🌌 Концепция игры
Мир разделен на две части: яркий город наверху и загадочное пространство под фонтаном. Когда человек бросает монетку, она падает к вам. Каждая монетка — это ключ к маленькой истории и конкретному поручению (уборка, доставка, готовка и другие).

## 🕹️ Режимы игры и Core Loop
* **Сюжетный режим:** Игрок ловит случайные монетки, исполняет желания и заполняет шкалу **«Счастья города»**. Каждое успешно выполненное задание навсегда открывает доступ к этой мини-игре в свободном режиме.
* **Свободный режим:** Библиотека уже исполненных желаний. Здесь можно переигрывать любимые мини-игры в любое время, оттачивая мастерство.

## ⚙️ Техническая информация
* **Движок:** Unity 6 (Версия 6000.3.6f1)
* **Язык:** C#
* **Платформа:** PC
* **Контроль версий:** Git + Git LFS

## ⌨️ Управление
* **Интерфейс и Хаб:** Мышь (выбор монетки, навигация в меню).
* **Мини-игры:**
    * `WASD` / `Стрелки` — Передвижение персонажа.
    * `Space` или `Enter` — Взаимодействие.

## 🚀 Установка для разработки
1.  Склонируйте репозиторий:
    ```bash
    git clone [https://github.com/ВАШ_ЛОГИН/Wishes-of-the-Fountain.git](https://github.com/ВАШ_ЛОГИН/Wishes-of-the-Fountain.git)
    ```
2.  Убедитесь, что установлен **Git LFS** для корректной загрузки медиа-файлов.
