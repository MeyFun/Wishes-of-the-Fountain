# Wishes of the Fountain

**Wishes of the Fountain** — это 2D-игра для ПК в жанре симулятора исполнения желаний. Игрок выступает в роли духа, живущего в глубинах городского фонтана. Собирая монетки, которые люди бросают в воду, вы узнаете об их сокровенных бытовых просьбах и тайно помогаете им воплотить их в жизнь.

## 🌌 Концепция игры
Мир разделен на две части: яркий город людей и таинственное «подмирье» под фонтаном. Каждая упавшая монетка запускает случайное событие. Ваша задача — выполнить серию мини-игр, чтобы сделать город чуточку счастливее.

## 🕹️ Игровой процесс (Core Loop)
1. **Ожидание:** Находимся в подмирье и ловим случайно падающую монетку.
2. **Задание:** Считываем желание (уборка подъезда, доставка посылки, готовка еды и др.).
3. **Действие:** Проходим мини-игру, соответствующую типу желания.
4. **Результат:** - **Успех:** Шкала «Счастья города» заполняется, а игра разблокируется в «Свободном режиме».
   - **Провал:** Счастье не растет, монетка не попадает в коллекцию.

## 🛠 Технологический стек
* **Движок:** Unity 6000.0.36f1
* **Язык программирования:** C#
* **Контроль версий:** Git + Git LFS (для хранения аудио и видео)
* **Целевая платформа:** PC (Windows)

## ⌨️ Управление
* **Меню и Хаб:** Мышь (навигация и выбор).
* **Мини-игры:**
    * `WASD` или `Стрелки` — Передвижение персонажа.
    * `Space` или `Enter` — Взаимодействие с объектами (поднять, использовать, активировать).

## 🚀 Установка и запуск
1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/MeyFun/Wishes-of-the-Fountain.git](https://github.com/MeyFun/Wishes-of-the-Fountain.git)
   ```
2. Убедитесь, что у вас установлен Git LFS, чтобы корректно загрузились медиа-файлы.
3. Добавьте проект в Unity Hub и откройте его версией 6000.0.36f1.
4. Дождитесь первичного импорта ассетов и генерации папки Library.

Проект выполнен в рамках лабораторной работы №5-6 на тему «Проектирование 2D-игры для ПК».
